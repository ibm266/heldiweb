# Legal documents

Drafts for the Heldi LTD store. **These are working drafts, not legal advice —
have a solicitor review them before publishing.**

| Document | Purpose |
|---|---|
| [terms-and-conditions.md](./terms-and-conditions.md) | Terms of sale — ordering, prices, delivery, liability |
| [privacy-policy.md](./privacy-policy.md) | UK GDPR privacy notice |
| [returns-refunds-policy.md](./returns-refunds-policy.md) | Cooling-off, sealed-goods exception, faulty items |
| [shipping-policy.md](./shipping-policy.md) | Delivery rates and times |
| [cookie-policy.md](./cookie-policy.md) | Cookies / local storage (currently none requiring consent) |

## Before publishing — fill in every `[TBC]`

- Company registration number (Companies House)
- VAT registration number (if/when registered)
- Contact email address (documents currently assume `hello@heldi.co.uk`)
- Dispatch time commitment in the shipping policy

## Where they go

1. Create a site page for each (e.g. `/legal/terms`, `/legal/privacy`, …) and
   link them in the site footer.
2. Add the same policies in Shopify admin (Settings → Policies) so checkout
   links to them.
3. Keep the shipping policy in sync with Shopify's actual shipping profile —
   Shopify is what customers are charged.
