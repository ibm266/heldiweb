# Generation prompts for Claude Design

Copy, adapt, and attach files from `references/` when using these templates.

## Master context block

Paste at the start of any Heldi design prompt:

```
Brand: Heldi, desi protein powder for home-cooked Indian food (dal, curry, raita, dahi). One pouch, whole family. Made in UK, launching autumn 2026.

Visual system:
- Colors: marigold #eda31d, ink blue #011246, cream #f8f0de, terracotta #a8432b
- Typography: Rozha One (headlines), Gelasio (body), NO sans-serif
- Style: warm editorial DTC, thick 3px ink borders, hard offset sticker shadows (6–8px, no blur), sticker cards, decorated Indian elephant line art
- Never: protein shakes, gym, blenders, "scoop", black #000, soft drop shadows

Attach reference images for colour and motif accuracy.
```

```

Food: dal tadka, jeera rice, cucumber raita, papad, home-cooked, not restaurant plating. Hands reaching to serve. No faces required (crop above shoulders).

No shaker bottles. No gym. No sans-serif text baked into image.
```

## Checklist before accepting output

- [ ] Ink is `#011246`, not black
- [ ] Marigold is `#eda31d`
- [ ] All type is serif (Rozha One + Gelasio)
- [ ] Shadows are hard offset, not soft blur
- [ ] Borders are 2–3px solid ink
- [ ] No shake/blender/gym
- [ ] Copy says spoon/stir, not scoop
- [ ] Single pouch SKU only
